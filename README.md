# Crab Grabber — Restaurant Website + QR Waitlist + SMS

> **Public disclaimer:** This repository is **not** the official website of any restaurant business.
> It is a personal/hobby project built by the site owner for learning and prototyping purposes.
> Names, branding, menu items, addresses, and other details may be placeholders or examples and should not be treated as official information.


This repository contains a simple static restaurant website prototype.

## Pages

- `public/index.html` — Home page
- `public/menu.html` — Menu page
- `public/about.html` — About page
- `public/contact.html` — Contact & location page

Open this file in a browser:

## Deployment

This site can be deployed using GitHub Pages by serving the `public/` folder.

Notes:

- Uses relative paths for compatibility with GitHub Pages
- Uses automatic cache-busting via GitHub Actions
- No backend or server required

## Purpose

This project is a personal static website prototype created for learning and demonstration purposes.
